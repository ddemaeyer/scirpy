scirpy.datasets.vdjdb

scirpy.datasets.vdjdb(cached=True)

Download VDJdb and process it into an AnnData object.

VDJdb [BVS+19] is a curated database of T-cell receptor (TCR) sequences with known antigen specificities.

Parameters
cached : boolbool (default: True)

If True, attempt to read from the data directory before downloading

Return type

AnnDataAnnData

Returns

An anndata object containing all entries from VDJDB in obs. Each entry is represented as if it was a cell, but without gene expression. Metadata is stored in adata.uns["DB"].
